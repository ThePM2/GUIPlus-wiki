# GUIPlus

*Create fully customizable GUIs for your Minecraft server ‚Äî no coding required.*

---

GUIPlus is a powerful GUI utility plugin that allows server administrators to build complex, interactive inventory menus through simple YAML configuration. From basic server menus to advanced banking systems with conditions and multi-scene navigation, GUIPlus handles it all.

> **New to GUIPlus?** Start with the [Getting Started](Getting-Started) guide to install the plugin and create your first GUI in minutes. Or jump straight to [Tutorials](Tutorials) for step-by-step walkthroughs of real-world server menus.

## Key Features

- **In-Game GUI Editor** ‚Äî Create and edit GUIs directly in-game with an intuitive editor interface
- **25 Click Event Types** ‚Äî Messages, commands, economy actions, teleportation, scene navigation, and more
- **11 Condition Types** ‚Äî Control item visibility and click execution with permission checks, balance requirements, placeholders, cooldowns, and more
- **Multi-Scene Support** ‚Äî Build multi-page GUIs with independent layouts and navigation
- **PlaceholderAPI Integration** ‚Äî Dynamic content that updates in real-time using any PAPI placeholder
- **Chat Fetcher System** ‚Äî Prompt players for input via chat, then use that input in click events
- **Player Data Storage** ‚Äî Save and retrieve per-player data fields accessible via placeholders
- **Custom Skull Textures** ‚Äî Use Base64 textures or dynamic player skins with `%self_base64%`
- **Economy Support** ‚Äî Give, remove, or set player balances with Vault integration
- **BungeeCord Support** ‚Äî Send players to other servers in your network
- **DeluxeMenus Converter** ‚Äî Automatically convert existing DeluxeMenus configurations
- **4 Inventory Types** ‚Äî Chest, Dispenser, Dropper, and Hopper
